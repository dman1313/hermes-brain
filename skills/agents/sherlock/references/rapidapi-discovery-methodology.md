# API Discovery Methodology — Rapid Financial Queries

## Query Structure for Finding New APIs

RapidAPI's search page is fully JS-rendered (Next.js). You cannot scrape it.
Instead, use these Google/GitHub patterns:

## Google Search Queries
```
site:rapidapi.com finance API free tier stock
site:rapidapi.com stock market API
"RapidAPI" "finance" "free" API stock
best free stock market API 2026
free stock options data API
financial news API free tier
```

## GitHub API Search
```
q=stock+market+api+free&sort=stars&order=desc
q=financial+data+api&sort=stars&order=desc
q=free+stock+api&sort=stars&order=desc
```
