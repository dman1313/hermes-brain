# Agent Memory Vault

Shared knowledge base for the multi-agent fleet (Hermes, Claude Code, Codex, Goose, Kimi, Kiro, Anti-Gravity).

## Locations

- **VPS (Hermes):** `/home/ubuntu/agent-memory` (Git clone)
- **Mac agents:** `/Users/dwayne-primeau/Library/Mobile Documents/com~apple~CloudDocs/Agent Memory/Coding`
- **GitHub:** `https://github.com/dman1313/agent-memory-coding`

## Key Files

| File | Purpose |
|------|---------|
| `NOW.md` | Live dashboard — auto-generated, never edit directly |
| `ACTIVITY.md` | Activity feed — prepend entries here |
| `DECISIONS.md` | Architectural decisions — append only |
| `MEMORY.md` | Index of all memories with [[wikilinks]] |

## Session Protocol (Hermes on VPS)

### Session Start
```
cd /home/ubuntu/agent-memory
git pull --rebase --autostash
cat NOW.md          # current fleet state
cat MEMORY.md       # index
# follow [[wikilinks]] for context
# prepend to ACTIVITY.md:
# YYYY-MM-DDTHH:MM:SSZ | hermes | session-start | project | description
git add -A && git commit -m "activity: session-start" && git push
```

### Session End
```
# prepend to ACTIVITY.md:
# YYYY-MM-DDTHH:MM:SSZ | hermes | session-end | project | summary
git add -A && git commit -m "activity: session-end" && git push
```

### Saving Memories
1. `git pull --rebase --autostash`
2. Create/update file in correct folder (User/, Feedback/, Project/, Reference/)
3. Frontmatter: name, description, metadata type
4. Add link in MEMORY.md index
5. Use [[wikilinks]] for cross-references
6. `git add -A && git commit -m "memory: <topic>" && git push`

## Activity Format

```
YYYY-MM-DDTHH:MM:SSZ | agent-name | event-type | project | description
```

### Event Types
`session-start`, `session-end`, `decision`, `blocker`, `blocker-resolve`, `milestone`, `handoff`, `note`

### Agent Names
`claude-code`, `codex`, `goose`, `kimi`, `kiro`, `hermes`, `antigravity`

## Important

- Always `git pull` before writing to avoid conflicts
- NOW.md is auto-generated by `build-context.sh` — never edit directly
- DECISIONS.md is append-only — never delete entries
- Auto-sync cron runs every 15 minutes on VPS
- Mac agents write directly to iCloud path (no git needed)
