# Exam Study Resource Builder

Create comprehensive, student-friendly, exam-focused study resource sets. Each resource must feel made for a specific student — not generic.

## Naming Convention

Every resource title follows a consistent pattern:

```
# [Series Name]: [Specific Resource Title]
```

Example: `# Look at this, Sarah: Motion and Forces One Page Review`

The series name creates identity. The subtitle makes each resource scannable.

## Study Priority Calibration

Before creating resources, establish priority weighting based on:
- Which units carry the most marks
- Which topics are most frequently tested
- Which concepts students find hardest
- Which topics combine calculations AND explanations

Present as a table with percentage and recommended study hours.

## Resource Set Architecture

Create resources in this order. Each must be complete and usable on its own.

### Tier 1: Core Resources

1. **Master Study Guide** — Overview of all units, key definitions, key equations, units, common question types, examiner traps, worked examples, practice questions, answer key, final revision checklist
2. **One-Page Review Sheets** — One per unit. Must-know definitions, equations, units, common traps, one model Q&A, three retrieval questions
3. **Examiner Traps** — For each trap: wrong answer, why it loses marks, correct answer, practice question with answer
4. **Step-by-Step Worked Examples** — Each follows strict structure (see below)
5. **Practice Test** — Sections: Quick Knowledge, Calculations, Graphs/Diagrams, Explanations, Examiner Trap Questions. Full mark-scheme-style answer key

### Tier 2: Supplementary Resources

6. **Flashcards** — Q&A format, short and testable
7. **Audio Review Script** — Natural spoken style, includes pause-and-answer moments, covers traps, ends with recap
8. **Slide Deck Outline** — Title, 3-5 bullets per slide, suggested visual, speaker notes, retrieval questions

### Tier 3: Video Scripts (if requested)

Each video covers ONE concept only. Use the Video Script Structure below.

## Worked Example Structure (mandatory for every worked example)

```
## Worked Example: [Topic]

### Question
### What the examiner is testing
### Step 1: Choose the equation or key idea
### Step 2: Check the units
### Step 3: Substitute or build the explanation
### Step 4: Calculate or write the answer
### Step 5: Final answer (with correct unit)
### Step 6: Examiner trap
```

## Video Script Structure (mandatory for every video)

```
# [Series Name]: [Concept Name]

## Video Length (5-10 minutes)
## Learning Goal
## Why This Matters for the Exam
## Opening Hook (short, clear, sets up the trap or key idea)
## Core Concept Explanation (short sections, no textbook paragraphs)
## Key Equations or Key Words
## Worked Example (full step-by-step)
## Common Mistakes (wrong answer, why it loses marks, correct answer)
## Exam Strategy (direct phrases: "First, check the units")
## Retrieval Practice (3-5 questions, then answers)
## 30-Second Recap
## Slide Deck Outline (title + bullets + visual + speaker notes per slide)
## Visual Suggestions
## On-Screen Keywords
```

## Style Rules

Use:
- Clear, calm, student-friendly, step-by-step, exam-focused, supportive
- Short paragraphs, tables, headings, numbered steps
- "This is the trap." / "Here is how to get the mark." / "Write it this way in the exam."
- "Check your units before calculating." / "The examiner wants these words."
- "Pause and try this yourself."

Avoid:
- "This is guaranteed." / "You will definitely get this question." / "This is easy."
- "Only memorize this." / "Don't worry about the rest."
- Childish tone, overly dramatic language, vague advice
- Unsupported claims about what will or won't appear on a test

Use safer exam language: "High priority" / "Very common" / "Often tested" / "Common exam pattern" / "Worth practising"

## Key Patterns for Science Subjects

### Unit Conversions (always check)
- g to kg (divide by 1000)
- cm to m (divide by 100)
- minutes to seconds (multiply by 60)
- kJ to J (multiply by 1000)

### The "Never Say" Pattern
For each topic, identify the phrase that loses marks and teach the correct replacement:
- "Energy is lost" -> "Energy is transferred to the thermal store of the surroundings"
- "Gravity" (for g) -> "Gravitational field strength"
- "Positive charges move" -> "Only electrons move"

### The Full Chain Pattern
For process questions (e.g., safety devices), teach every step in sequence. Students lose marks by skipping steps.

```
# [Series Name]: [Resource Title]
## What this helps you do
## What you need to know
## What examiners often ask
## What students often get wrong
## Step-by-step example
## Try it yourself
## Check your answers
## Final recap
```

## Practice Question Rules

Each set includes:
- 3 quick recall questions
- 3 calculation questions
- 2 explanation questions
- 1 examiner trap question
- 1 challenge question

Answers must be step-by-step and mark-scheme style. For written answers, show exact phrases the student should use.

## Pitfalls

- Do not combine multiple difficult concepts into one video
- Do not invent past paper dates or mark scheme wording unless clearly in the source material
- Do not give vague answers in the answer key — show working
- Do not overload slides — max 5 bullets per slide
- Do not skip the "Examiner trap" step in worked examples
- Do not forget to include units in final answers
- Always convert units before substituting into equations
- Always teach the "full chain" for process/explanation questions
- Do not create generic resources — every resource must feel made for the specific student
