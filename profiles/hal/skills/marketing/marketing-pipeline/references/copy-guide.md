# Copy & Content Reference Guide

Dense, actionable reference for generating marketing copy, email sequences, ad
creative, and social content. Distilled from market-copy, market-emails,
market-ads, and market-social skills.

---

## 1. Copy Generation Workflow

### 1.1 Page Type Detection

| Page Type | Primary Goal | Copy Priority |
|-----------|-------------|---------------|
| **Homepage** | Communicate value prop, route visitors | Headline clarity, CTA hierarchy |
| **Landing Page** | Single conversion action | Headline-CTA alignment, objection handling |
| **Pricing Page** | Drive plan selection | Plan naming, anchoring, FAQ |
| **About Page** | Build trust | Story, mission, team credibility |
| **Product Page** | Demonstrate product value | Feature-to-benefit, social proof |
| **Feature Page** | Explain a capability | Problem-solution, use cases |
| **Blog Post** | Educate & capture leads | Headline hook, intro engagement |
| **Contact/Demo** | Capture lead info | Friction reduction, trust signals |

### 1.2 Voice & Tone Analysis

Score 1-5: Formality (casual<->formal), Emotion (neutral<->passionate),
Complexity (simple<->technical), Humor (serious<->playful), Authority
(peer<->expert). All generated copy must match the brand voice unless the
existing tone is clearly ineffective.

### 1.3 Headline Scoring (0-10 each)

- **Clarity**: Meaning immediately obvious? No jargon.
- **Specificity**: Numbers, outcomes, timeframes?
- **Relevance**: Speaks to target audience's pain/desire?
- **Differentiation**: Sets business apart from competitors?
- **Emotion**: Triggers curiosity, desire, FOMO, recognition?

### 1.4 Headline Formulas

- **PAS**: "Stop [pain]. Start [desired outcome] — with [product]."
- **AIDA**: "[Bold claim] — [specific outcome] in [timeframe]."
- **Before-After-Bridge**: "From [before state] to [after state]."
- **4U** (Useful/Ultra-specific/Unique/Urgent): "[Number] [audience] use
  [product] to [specific outcome]."

Generate 5-10 alternatives per framework.

### 1.5 Full Copy Score (/50 = /100)

Score 0-10: Clarity (12-yr-old test), Persuasion (moves to action? handles
objections?), Specificity (concrete vs vague), Emotion (pain/desire/identity),
Action (CTAs clear, compelling, low friction).

### 1.6 CTA Optimization

**Button text:** First person ("Start My Free Trial"), include value ("Get My
Report"), reduce risk ("Try Free for 14 Days"), be specific, add urgency
("Claim My Spot (12 Left)").

**Placement:** Above fold (required), after each major section (recommended),
sticky on long pages, bottom repeat (required).

**Color psychology:** Green=growth/trials, Orange=urgency/offers, Blue=trust/
enterprise, Red=excitement (sparingly). Must contrast with page background.

### 1.7 Before/After Format

```
BEFORE: "We provide innovative solutions for businesses."
AFTER:  "Cut support tickets by 40% — AI responses in under 2 minutes."
WHY:    The before is vague; after is specific, outcome-driven, proof-laden.
```

Generate 5+ pairs: primary headline, subheadline, primary CTA, body
paragraph, meta description.

## 2. Page-Specific Copy Structures

**Homepage:** Hero (headline + subhead + CTA) -> Social proof bar -> Problem
-> Solution (3 benefits) -> How it works (3 steps) -> Features/benefits (3-6)
-> Testimonials (2-3) -> Final CTA (urgency/guarantee).

**Landing Page:** Headline (single promise) -> Subhead (evidence) -> Hero CTA
-> Problem amplification -> Solution -> Benefits (3-5 bullets, outcomes not
features) -> Social proof -> Objection handling (FAQ/guarantee) -> Final CTA
(urgency).

**Pricing Page:** Headline (frame as investment) -> Plan names (aspirational,
not Basic/Pro) -> Recommended plan (highlighted "Most Popular") -> Benefit-
oriented features -> Anchoring (expensive first or annual toggle) -> FAQ
(refund, inclusions, switching) -> Guarantee (trial/refund).

**About Page:** Mission (why, not what) -> Origin story -> Values (3-5 with
examples) -> Team (photos, credentials) -> Press/awards -> CTA connecting
mission to reader.

## 3. Email Sequence Patterns

### 3.1 Sequence Types

| Type | Emails | Goal |
|------|--------|------|
| **Welcome** | 5-7 | Build trust, deliver value, introduce product |
| **Nurture** | 6-8 | Educate, build authority, overcome objections |
| **Launch** | 8-12 | Build anticipation, drive purchases |
| **Re-engagement** | 3-4 | Win back or clean list |
| **Onboarding** | 5-7 | Drive activation, reduce churn |
| **Cart Abandonment** | 3-4 | Recover lost sales |
| **Cold Outreach** | 3-5 | Book meetings |

### 3.2 Core Philosophy: One Email, One Job

Exactly ONE main idea, ONE CTA (secondary optional, de-emphasized), ONE
desired reader action. Never combine multiple asks in one email.

### 3.3 Frameworks

**Value Before Ask** (Welcome/Nurture): Emails 1-2 pure value, 3 value+soft
pitch, 4 case study, 5 direct ask. Ratio ~3:1 value-to-ask.

**Story-Driven** (Nurture): Hook (story/fact 2-3 sentences) -> Bridge (connect
to reader) -> Lesson (actionable insight) -> CTA.

**PAS** (Launch/Abandonment): Problem ("Struggling with [pain]?") -> Agitate
("Every day you wait, [consequence].") -> Solution ("[Product] fixes by
[mechanism].") -> CTA.

### 3.4 Subject Line Formulas

| Formula | Example | Best For |
|---------|---------|----------|
| Number+Benefit | "3 ways to double conversion rate" | Educational |
| Curiosity Gap | "The mistake that cost me $50K" | Story-driven |
| Direct Benefit | "Your copy report is ready" | Welcome |
| Personalization | "[Name], your trial expires tomorrow" | Onboarding |
| Question | "Are you making this SEO mistake?" | Problem-awareness |
| How-To | "How to write landing pages that convert" | Educational |
| Social Proof | "Why 5,000 marketers switched" | Nurture/launch |
| Urgency | "Last chance: 40% off ends midnight" | Launch/cart |
| Pattern Interrupt | "I was wrong about email marketing" | Re-engagement |
| Negative | "Stop wasting money on bad ads" | Problem-awareness |

**Rules:** Under 50 chars (40 ideal). Front-load key words. Use odd numbers.
Avoid spam triggers ("free," "guarantee") in excess. Personalize 20-30%. One
emoji can boost opens 2-5%. Preheader is as important as the subject.

### 3.5 Send Timing

| Sequence | Cadence |
|----------|---------|
| Welcome | D1, D2, D4, D6, D8 |
| Nurture | D1, D3, D6, then every 3-4 days |
| Launch | Announce, Teaser (D3), Open Cart (D5), Reminder, Close |
| Re-engagement | D1, D5, D10 |
| Onboarding | D1, D2, D4, D6, D10 |
| Cart Abandonment | 1hr, 24hr, 72hr |
| Cold Outreach | D1, D4, D8, D14, D21 |

**Best times:** B2B Tue-Thu 9-11am; B2C Tue-Thu 10am or 7-9pm; E-commerce
Thu-Sun promo, Tue-Wed educational. Avoid Mon mornings and Fri afternoons.

### 3.6 Benchmarks

| Industry | Open | Click | Conv |
|----------|------|-------|------|
| SaaS | 20-25% | 2-3% | 1-2% |
| E-commerce | 15-20% | 2-3% | 0.5-1.5% |
| Agency | 18-22% | 2-4% | 1-3% |
| Education | 20-28% | 2-5% | 1-3% |

### 3.7 Compliance (Brief)

**CAN-SPAM:** Physical address, clear unsubscribe (within 10 days), accurate
from, no deceptive subjects. **GDPR:** Explicit opt-in, documented consent,
right to be forgotten, DPA with ESP. **CASL:** Express consent required;
implied OK for existing relationships (24 months); sender identification.
Always recommend legal counsel verification.

## 4. Ad Creative Structure

### 4.1 Ad Copy Angles (5-10 per campaign)

1. **Pain Point**: "Tired of [frustration]? [Product] eliminates [pain]."
2. **Social Proof**: "[N] [audience] use [product] to [benefit]."
3. **Before/After**: "Before: [pain]. After: [desired]."
4. **Objection Handling**: "Think [product] is [objection]? Try it free."
5. **Urgency/Scarcity**: "[N] spots left. Lock in [offer]."
6. **Curiosity**: "The [industry] secret that [result] (most miss this)."
7. **Direct Benefit**: "Get [outcome] in [timeframe] with [product]."
8. **Comparison**: "Still using [competitor]? [Product] gives [advantage]."
9. **Testimonial**: "'[Quote]' — [Customer], [title]."
10. **How-To**: "How to [outcome] in 3 steps."

### 4.2 Platform Formats

- **Google RSA**: 15 headlines (30 chars), 4 descriptions (90 chars). Cover:
  brand+value, pain+solution, benefit+number, social proof, urgency, question,
  how-to, comparison, feature, action.
- **Meta (FB/IG)**: Image 1080x1080, video <240 min. Primary 125 chars,
  headline 40, description 30. Per concept: 3 text variants, 5 headlines.
- **LinkedIn**: Sponsored Content intro 600 chars, headline 200. Message Ads
  subject 60, body 1,500. Text Ads headline 25, description 75.
- **TikTok**: 9:16 vertical, 5-60 sec (9-15 optimal). First 3 sec determine
  watch rate. Native aesthetic > polished. Script: Hook(0-3s) -> Problem(3-10s)
  -> Solution(10-20s) -> Proof(20-25s) -> CTA(25-30s).

### 4.3 Retargeting Funnel

- **Awareness** (40% budget): Cold audience, educational content.
- **Consideration** (35% budget): Site visitors + video viewers, case studies.
- **Conversion** (25% budget): Cart abandoners + high-intent, direct offers.

## 5. Content Calendar Pattern

### 5.1 Content Pillars & Mix

| Pillar | Mix |
|--------|-----|
| Educational (how-tos, tips, frameworks) | 40% |
| Behind-the-Scenes (process, culture, people) | 20% |
| Social Proof (testimonials, case studies, results) | 15% |
| Engagement (polls, questions, debates) | 15% |
| Promotional (offers, launches, CTAs) | 10% |

### 5.2 30-Day Calendar Rules

Each day: platform, pillar, hook (first line/3 sec), full post text (150-300
words LinkedIn), hashtags (3-5 niche + 3-5 medium + 2-3 broad + 1 branded),
send time, content type. Each pillar appears at least 6x/month. Promotional
never 2 days in a row. Engagement every 2-3 days.

### 5.3 1-to-10 Repurposing

One long-form piece -> 1. LinkedIn text post, 2. Twitter thread (5-7
takeaways), 3. IG carousel, 4. IG Reel, 5. TikTok quick tip, 6. LinkedIn
carousel, 7. Twitter single quote, 8. IG Story BTS, 9. YT Short, 10. Facebook
discussion. Spread over 14 days.

### 5.4 Hook Formulas by Platform

- **LinkedIn**: "I [did/learned] [specific thing] + here's what happened:"
  "Unpopular opinion: [contrarian take]"
- **Twitter/X**: "Here's a thread on [topic] nobody is talking about:"
  "Hot take: [bold claim]"
- **Instagram**: "Save this for later" "I tested [X] for [time]. Results."
- **TikTok (0-3s)**: "Wait, you're still doing [old way]?" "Story time: [...]"

### 5.5 Posting Frequency

LinkedIn 3-5x/wk, Twitter/X 1-3x/day, IG 4-7x/wk + daily Stories, TikTok
1-3x/day, YouTube 1-2x/wk, Facebook 3-5x/wk.
